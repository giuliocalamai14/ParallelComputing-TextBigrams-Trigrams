Per eseguire da Terminale:

	-Sequential
		1) Creare directory con all'intero:
			-Main.java
			-Sequential.java
			-text.txt
			-textBig.txt
		2) Recarsi all'interno della directory creata
		3) $ javac Main.class -d out
		4) $ cd ./out
		5) java Main

	-Parallel
		1) Creare directory con all'intero:
			-Main.java
			-Parallel.java
			-text.txt
			-textBig.txt
		2) Recarsi all'interno della directory creata
		3) $ javac Main.class -d out
		4) $ cd ./out
		5) java Main
